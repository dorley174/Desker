# Desker Frontend (offline mock edition)

Улучшенный frontend для системы бронирования рабочих мест, адаптированный под сценарии из `project.md` и запускаемый **без backend**.

## Что внутри

- авторизация и регистрация по инвайт-коду;
- восстановление пароля через встроенный демо-код;
- карта мест, фильтры, почасовое бронирование и walk-in;
- история бронирований и профиль;
- админ-панель с аналитикой загрузки;
- встроенные mock-заглушки вместо API.

## Запуск

```bash
npm install
npm run dev
```

## Тестовые аккаунты

- `user@desker.io` / `user123`
- `admin@desker.io` / `admin123`

## Инвайт-коды

- `JOIN2026` — сотрудник
- `ADMIN2026` — администратор

## Как работают данные

Приложение не обращается к серверу. Все данные хранятся в `localStorage` браузера и сохраняются между перезагрузками.

Восстановление пароля в демо-режиме использует код `123456`.
